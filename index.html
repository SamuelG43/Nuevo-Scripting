Instrucciones de configuración

Hola, para configurar tu pagina web, sigue los siguientes pasos:

  1. Debes tener un archivo index.html en la raiz de tu repositorio en la rama principal main, en este caso es este acrhivo
  2. Subir el archivo por medio de un commit al repositorio
  3. Activar la opción de GitHub Pages en la configuración de tu repositorio
  4. Listo, tu pagina web estará disponible en la siguiente dirección: https://nombre_usuario.github.io/nombre_repositorio/
  5. En el paso anterior, reemplaza nombre_usuario por tu nombre de usuario de GitHub y nombre_repositorio por el nombre de tu repositorio
  6. Si deseas cambiar el contenido de tu pagina web, solo debes modificar el archivo index.html y subirlo por medio de un commit a tu repositorio
  7. La url es publica y la puedes compartir con otros usuario de internet
